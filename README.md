Sanity + Vue.js

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Run the studio

To start the development server for Sanity Studio, run this command within the studio folder:

```
sanity start
```
